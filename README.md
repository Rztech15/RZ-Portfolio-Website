## 📌 About This Project

A fully responsive personal portfolio website for **Muhammad Ramzan** — BS Mathematics student at CASPAM, Bahauddin Zakariya University Multan. Built with a dark navy + pink-purple gradient aesthetic inspired by modern developer portfolios.

This portfolio showcases AI projects, data analysis work, web development skills, and academic background — all in one clean, professional site.

---

## ✨ Features

- 🌑 **Dark Navy Theme** — deep dark background with pink-purple gradient accents
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- 🔢 **Stats Section** — highlights key achievements at a glance
- 📊 **Animated Skill Bars** — visual progress bars for each technology
- 🕐 **Experience Timeline** — clean vertical timeline layout
- 🗂️ **Projects Grid** — 6 projects with category tags and live links
- 📬 **Contact Form** — name, email, subject, message fields
- ⚡ **Smooth Scroll Navigation** — fixed navbar with anchor links
- 👑 **RZ Brand Identity** — personal 👑🖤 branding throughout

## 🔧 How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/rztech15/RZ-Portfolio-Website.git

# 2. Open in browser
cd RZ-Portfolio-Website
open index.html
```

No dependencies. No build step. Just open `index.html` in any browser. ✅

## 🌐 Deployment

